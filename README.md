# CLI-Program--MainMenu-for-Customer-and-Admin-Operations
this is a cli java program <br />
 having smooth menu he can choose from lots of functions as tasked, an admin access is available. <br />
Note: admin pin is inside program.java <br />
<br />
Main Menu <br />
The application displays a main menu with options for customers and admins. <br />
Customers can create a ticket by entering a ticket number, description, and selecting a priority (stored in separate ArrayLists). <br />
Priority options include LOW, MEDIUM (default), and HIGH (stored in an Array). <br />
Admins must enter a valid PIN to access the admin submenu. <br />
An option to exit the application is available. <br />
Customer Operations <br />
Customers can input a ticket number and description. <br />
Customers can select a priority from the available options. <br />
Tickets are stored in ArrayLists for ticket numbers, descriptions, and priorities. <br />
Admin Operations <br />
Admins must enter a correct PIN to access the submenu. <br />
The admin submenu allows viewing a ticket's description and priority by entering the ticket number. <br />
Admins can update the ticket description or exit the submenu. <br />
General Functionality <br />
All menus and submenus include an option to exit. <br />
The application validates inputs (e.g., valid PIN for admin access, valid priority selection). <br />
